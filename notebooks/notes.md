Penelitian ini membandingkan tiga arsitektur deep learning, yaitu Deep Neural Network (DNN), 1D-Convolutional Neural Network (1D-CNN), dan Temporal Convolutional Network (TCN), untuk mendeteksi kejadian awan Cumulonimbus berbasis data indeks stabilitas udara atas. Target klasifikasi yang digunakan adalah kejadian Cumulonimbus dalam rentang 12 jam setelah waktu pengamatan radiosonde (cb_h12). Data dibagi berdasarkan waktu, yaitu periode 2020–2022 sebagai data latih, tahun 2023 sebagai data validasi, dan tahun 2024 sebagai data uji. Skema ini dipilih untuk menghindari data leakage dan mensimulasikan kondisi prediksi operasional terhadap data pada periode yang lebih baru.

Sebelum pemodelan, fitur numerik dari data sounding dipilih setelah mengeluarkan kolom identitas waktu dan label. Nilai tidak valid seperti -9999, 9999, serta nilai tak hingga dikonversi menjadi missing value. Fitur dengan proporsi missing value lebih dari 40% dikeluarkan dari pemodelan. Missing value pada fitur yang dipertahankan diisi menggunakan median berdasarkan data latih, kemudian dilakukan standardisasi menggunakan StandardScaler. Seluruh parameter preprocessing hanya diestimasi dari data latih dan diterapkan pada data validasi serta data uji untuk mencegah kebocoran informasi.

Model DNN menggunakan input tabular satu waktu pengamatan radiosonde dan terdiri atas tiga lapisan tersembunyi dengan 128, 64, dan 32 neuron. Setiap lapisan menggunakan aktivasi ReLU, batch normalization pada dua lapisan awal, serta dropout untuk mengurangi overfitting. Model 1D-CNN dan TCN menggunakan input sekuensial berupa tiga pengamatan radiosonde berurutan. Model 1D-CNN terdiri atas dua lapisan Conv1D dengan 64 dan 32 filter, sedangkan model TCN menggunakan tiga residual block dengan dilation rate 1, 2, dan 4 untuk menangkap ketergantungan temporal. Seluruh model menggunakan sigmoid pada lapisan output, binary crossentropy sebagai fungsi loss, optimizer Adam dengan learning rate 0,001, batch size 32, dan maksimum 300 epoch dengan early stopping.

Evaluasi model dilakukan pada data uji tahun 2024 menggunakan metrik accuracy, Probability of Detection (POD/Recall), False Alarm Rate (FAR), F1-score, Area Under Curve (AUC), dan confusion matrix. Metrik POD digunakan untuk menilai kemampuan model dalam mendeteksi kejadian CB yang benar-benar terjadi, sedangkan FAR digunakan untuk mengukur proporsi prediksi CB yang merupakan false alarm. Kombinasi metrik tersebut digunakan agar penilaian model tidak hanya berfokus pada akurasi, tetapi juga mempertimbangkan keseimbangan antara deteksi kejadian dan kesalahan alarm.
